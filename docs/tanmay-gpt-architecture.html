01Product Overview

Three creator tools, one persona core. Every tab talks to the same knowledge store and persona engine, but with a different prompt strategy, retrieval shape, and output schema.

Tab 1, Content Generation

Input: content idea. Output: script + description written the way Tanmay would approach the topic.

Tab 2, Ad Generation

Input: product, duration, cast, celebs. Output: full ad script + strategic rationale in Tanmay's branded-content voice.

Tab 3, Q&A

Input: any question. Output: Tanmay-grounded answer with citations, or an honest "he has never talked about this".

Design Principle

Keep "know Tanmay" (knowledge store) strictly separated from "act as Tanmay" (persona engine). The knowledge store is retrieval-only and factual. The persona engine is stylistic and opinionated. Mixing them couples data quality with voice quality, which makes debugging a nightmare.

Persona Prompt Anatomy

The system prompt is a living document, version-controlled like code, A/B tested like a feature flag. It has six sections:

  1. Identity anchor. Who Tanmay is, what he stands for, the tone ceiling and floor.
  2. Voice patterns. Sentence rhythm, Hinglish ratio, typical openings, callback humor, self-deprecation cadence.
  3. Topic posture. How he engages with AI, mental health, creator economy, cricket, finance curiosity, and what he avoids.
  4. Format rules. Per-tab output schema (script vs ad vs answer).
  5. Retrieved context. Ranked chunks with source metadata.
  6. Style exemplars. 3 to 5 short high-voice snippets for few-shot anchoring.

Why the Brand-Safety Gate Matters

If the user inputs a product in a category Tanmay has publicly refused (say, real-money gaming or sketchy crypto), the system should flag it before generating. This protects both the licensing deal and the user's credibility.

09Implementation Phases, Week by Week

Ten phases, roughly one per week for a solo build or parallelizable to 4-6 weeks with a small team.

PHASE 00

Legal Foundation

1 week, non-negotiable

Engage an IP lawyer. Draft a licensing agreement covering training data use, likeness rights, revenue share, territory, term, termination, and off-limits topics. Get signatures before a single scraper runs.

PHASE 01

Data Collection

1.5 weeks

Build the source registry YAML. Write and run platform-specific scrapers. Store raw payloads in R2 with manifest files. Target volume: 500+ hours of audio/video, 2000+ posts. De-dupe by URL hash. Checkpoint every 50 items.

PHASE 02

Transcription & Diarization

1 week

Batch-submit all audio/video to Deepgram with diarization enabled. Post-process to isolate Tanmay's speaker-label across the corpus. Store aligned transcripts (word + timestamp) so the UI can cite back to the exact second of a clip.

PHASE 03

Chunking, Tagging, Embedding

1 week

Semantic chunker that splits at topic shifts, not fixed sizes. Run a Claude Haiku pass over every chunk to tag: topic, format, register (roast, reflective, informative, comedic), language mix, named entities, sentiment. Embed with Voyage-3, upsert into Qdrant with full metadata. Build a separate "style-exemplar" collection for few-shot prompting.

PHASE 04

Persona Engineering

1 week

This is craft work, not engineering. Hand-analyze 50 to 100 prime Tanmay pieces to extract voice patterns. Write the persona system prompt. Validate via blind test: mix 20 generated outputs with 20 real Tanmay pieces, have Tanmay fans try to tell them apart. Target accuracy near 50% (indistinguishable).

PHASE 05

Backend, RAG Core + Tab 1

1.5 weeks

FastAPI scaffold. LangGraph RAG orchestrator. Implement POST /generate/content end to end with streaming. Plug in Langfuse tracing. Ship Tab 1 as the MVP because it is the cheapest to validate and the most forgiving of imperfect retrieval.

PHASE 06

Tab 2, Ad Generation

1 week

Structured form schema. Duration-to-wordcount validator. Brand-safety gate that flags category conflicts. Retrieval filter limited to ads and branded content. Structured JSON output enforced via tool use. Export formats: Markdown, PDF, Fountain (screenplay format).

PHASE 07

Tab 3, Q&A with Citations

1 week

Multi-query retrieval with paraphrase expansion. Confidence threshold gate. Citation verifier that maps every factual claim back to a source chunk and strips unverified claims. Clickable source UI with timestamp deep-links for YouTube and post URLs for text platforms.

PHASE 08

Frontend

parallel, 2 weeks

Next.js App Router with one route per tab. Streaming via Vercel AI SDK. Shared history sidebar. Copy, export, regenerate-with-different-tone. Tone dial component. Responsive design, mobile-first.

PHASE 09

Evals & Red-Teaming

1 week

Golden set of 100 prompts per tab with expert ratings. Automated style-match score using embedding similarity to the real Tanmay corpus. Red-team for off-brand outputs, defamation risk, sponsor conflicts, and sensitive topics. Any failure patterns feed back into persona prompt v2.

PHASE 10

Private Beta + Launch

2 weeks

Invite 50 creators, instrument every generation with Langfuse, daily prompt iteration. Pricing tiers live. Public launch with a launch video (ideally starring Tanmay himself).

11Extra Features

Features that turn this from a clever demo into a product people pay for.

01, Source-Grounded Citations

Every generation shows the chunks it drew from. Click a citation, it opens the original YouTube clip at the exact timestamp or the original tweet. This is trust, defensibility, and a differentiator all in one.

02, Tone Dial

Sliders for roast-level, chaos, depth, and Hinglish ratio. Lets creators tune output without rewriting prompts. Each slider maps to a specific prompt variable that gets injected into the composer.

03, Voice Output (Premium)

ElevenLabs voice clone (licensed, watermarked). Generated scripts become audio previews. Premium-tier only, with per-minute metering.

04, Why This Works Panel

Alongside every generation, a short meta-explanation of which Tanmay techniques were applied (callback humor, self-deprecation, pattern interrupt, audience misdirection). Turns the product into a creator-education tool.

05, Timeline Explorer

"Show me how Tanmay's take on AI evolved from 2019 to 2026." Pulls from the corpus chronologically with a visual timeline. Shareable, viral-friendly, and cheap to build on top of existing metadata.

06, Collaboration Mode

Two users co-edit a script in real time. AI suggestions inline via Tiptap + Yjs (CRDT). Useful for writer-creator pairs.

07, Brand-Safety Guardrails

Auto-flag if the product category in Tab 2 conflicts with things Tanmay has publicly refused (gambling, misleading financial products, etc.). Maintained as a rules file + semantic match against his historical refusals.

08, Prompt Template Library

Community-submitted prompt packs, upvoted. Starter templates like "Diwali brand campaign", "Roast my startup pitch", "Reaction to a trending clip". Templates are composable with the tone dial.

09, Chrome Extension

Highlight any text on the web, right-click "What would Tanmay say about this". Opens a side panel with the Tab 3 response. Great acquisition surface.

10, Creator Analytics Loop

If the user posts a generated script, a webhook or manual log feeds the performance data (views, retention) back. Over time you learn which prompt patterns actually move metrics, which is a moat.

11, Agency API Access

REST API for ad agencies pitching Tanmay-style campaigns. Priced per call, gated by a stricter brand-safety filter. This is likely the highest-ARPU channel.

12, Multi-Creator Platform (V2)

The same architecture generalizes. Once it works for Tanmay, the pipeline can onboard any licensed creator. The product becomes a marketplace: licensed creators get a rev-share, users get many voices. This is the actual business.

13, Prompt-to-Storyboard

For Tab 2 ads, auto-generate a storyboard using an image model for each scene direction. Creators go from idea to shootable visual brief in one click.

14, Community Remix Feed

Public generations (opt-in) go to a feed where other users can remix, vote, and share. Creates social proof and organic discovery.

15, Weekly Tanmay Digest

Automated weekly email: "Here is what new content Tanmay published and what the GPT thinks are the key patterns in it." Keeps users engaged without requiring a daily habit.

12Risks & Guardrails

Legal: Building this without a signed license from Tanmay or his team is the single biggest risk. Publicity rights, likeness, and platform ToS all apply. Phase 00 is not optional.
Hallucination in Tab 3: Users will ask things Tanmay has never discussed. The confidence gate + citation verifier must be strict. Better to refuse than to fabricate.
Sensitive topics: Politics, religion, personal relationships, mental-health advice. The persona prompt must hard-refuse these even if retrieval returns relevant chunks. This protects Tanmay's reputation, which is literally the product.
Cost runaway: LLM spend scales linearly with success. Rate limits, per-user quotas, aggressive caching of common queries, and tiered pricing are non-optional.
Stale corpus: Tanmay keeps creating. Weekly re-crawl via Celery Beat. Monitor embedding drift. Without this, the system slowly falls behind his current voice.
Voice clone abuse: If you ship ElevenLabs voice, watermark every audio output, require consent on every generation, and maintain a takedown pipeline for misuse.
